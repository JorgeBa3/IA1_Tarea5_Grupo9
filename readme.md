##  Descripción del bot

Bot de Telegram desarrollado en Python con la librería `python-telegram-bot`.  
Funciona como mini asistente del proyecto de IA1, proporcionando información básica del equipo y el proyecto.

El proyecto consiste en un **Sistema de Inferencia Lógica con Prolog** — un proyecto de Inteligencia Artificial que modela bases de conocimiento usando Prolog, con hechos, reglas y predicados recursivos para resolver problemas de inferencia lógica.

##  Lista de comandos

| Comando | Descripción |
|---------|-------------|
| `/hola` | Saluda al usuario y muestra los comandos disponibles |
| `/hora` | Muestra la fecha y hora actual del servidor |
| `/contacto` | Muestra correo, Telegram y GitHub del grupo |
| `/proyecto` | Muestra el nombre, descripción e integrantes del proyecto |
